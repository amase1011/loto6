# ロト6分析 v2.1

ロト6専用の個人利用向けWebアプリです。ロト7機能はありません。

## 主な機能

- 棒グラフ・折れ線グラフ
- ホット／コールドナンバー
- 直近10・30・50・100・300回／全期間分析
- 出現率・未出現期間などの並び替え
- 独自統計スコアによる参考候補
- 条件付き候補生成
- 分析画面のPNG保存（非対応ブラウザでは簡易レポート保存）
- PWA対応
- 宝くじ公式サイトのロト6購入ボタン
- 保存済みデータを利用したオフライン表示

## Render

Build Command:

```text
echo "No build step"
```

Start Command:

```text
node server.js
```

既存のRenderサービスはGitHubへコミットすると自動更新されます。

## 注意

- 過去データの統計であり、当選を予測・保証しません。
- 参考スコアは実際の当選確率ではありません。
- データ元の利用条件上、個人利用向けです。
